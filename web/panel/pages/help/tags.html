Теги пользовательских команд

  • Twitch
  • Discord
(#) генерирует случайное число от 1 до 100 –
(# ?) генерирует случайное число из заданного в теге диапазона (# 1,250)
(1), (2), (3), … (9) подставляет указанное после команды значение (или до 9 значений) –
(1=?) подставляет заданный в теге другой тег без скобок (опционально: указанное после команды значение) (1=random)
(alert ?) воспроизводит заданную в теге анимацию GIF, WebM или MP4 (опционально: с заданными длительностью, в секундах, громкостью от 0.1 до 1.0, стилем CSS и текстом) (alert banana.gif,15,0.8,color:white;font-family:Helvetica;font-weight:600;font-size:28px;,(sender) передаёт всем привет!)
(adminonlyedit) запрещает редактировать команду пользователям с уровнем доступа ниже Администратора –
(age) показывает стаж в Twitch вызвавшего команду пользователя (опционально: указанного после команды пользователя) –
(baresender) показывает имя вызвавшего команду пользователя в нижнем регистре –
(channelname) показывает имя стримера (владельца канала) –
(code=?) генерирует случайный код с заданным в теге количеством символов (code=8)
(command ?) вызывает заданную в теге команду (тег должен быть в начале отклика и без «!» перед именем команды) (command topclip)
(commandcostlist) показывает стоимость всех платных команд –
(count) показывает количество вызовов команды (счётчик) –
(currenttime ?) показывает текущее время в заданных в теге часовой зоне (см. подраздел «Часовые зоны») и формате времени (currenttime Europe/Moscow, HH:mm:ss)
(customapi ?) вызывает данные по заданной в теге ссылке API (customapi http://not.real.com/joke.php?name=$1)
(customapijson ?) вызывает данные в формате JSON по заданной в теге ссылке API (customapijson https://api.apixu.com/v1/current.json?key=ваш_ключ&q=$1&lang=ru {Температура в г. } location.name {: } current.temp_c { °C})
(downtime) показывает количество времени, прошедшее с окончания последнего стрима –
(echo) возвращает указанное после команды сообщение от имени бота (осторожно: с тегом также могут использоваться такие команды, как /timeout и /ban) –
(encodeurl ?) переводит пробелы в запросе, указанном после команды, в URL-кодировку (может использоваться внутри такого тега, как (customapi), а также сам содержать в себе такие теги, как (echo) и др.) (encodeurl https://google.com/search?q=(echo))
(followage) показывает стаж на канале вызвавшего команду пользователя (опционально: указанного после команды пользователя), в сутках –
(followdate) показывает дату фолловинга канала вызвавшего команду пользователя (опционально: указанного после команды пользователя) –
(follows) показывает количество фолловеров канала –
(game) показывает название текущей игры стрима –
(gameinfo) показывает название текущей игры стрима и его продолжительность –
(gameonly=?) разрешает команде работать только во время заданной в теге игры (gameonly=Dota 2)
(gamesplayed) показывает список игр, сыгранных за текущий стрим –
(hours) показывает стаж на канале вызвавшего команду пользователя (опционально: указанного после команды пользователя), в часах –
(keywordcount) показывает количество срабатываний триггера, в отклике которого тег установлен (актуально для раздела «Триггеры») –
(lasttip) показывает информацию о последнем донате –
(offlineonly) не разрешает команде работать во время стрима –
(onlineonly) разрешает команде работать только во время стрима –
(pay) показывает вознаграждение, выплачиваемое за вызов команды –
(playsound ?) воспроизводит заданный в теге звук (для этого необходимо добавить в OBS в качестве источника BrowserSource соответствующую ссылку – см. в настройках раздела «Звуки») (playsound airhorn)
(playtime) показывает продолжительность текущей игры –
(pointname) показывает пользовательское название поинтов в родительном падеже –
(points) показывает количество поинтов пользователя, вызвавшего команду, без названия поинтов –
(pointtouser) показывает имя пользователя со стрелкой на отклик команды, в случае если оно указано после команды –
(price) показывает плату, взимаемую за вызов команды –
(random) показывает случайное имя пользователя из чата –
(randomrank) показывает случайное имя пользователя из чата с упоминанием его ранга –
(readfile ?) показывает первую строку из заданного в теге текстового файла, лежащего в папке «addons» бота (readfile имя_файла.txt)
(readfilerand ?) показывает случайную строку из заданного в теге текстового файла, лежащего в папке «addons» бота (readfilerand имя_файла.txt)
(@sender) показывает имя пользователя, вызвавшего команду, с символом «@» перед именем –
(sender) показывает имя пользователя, вызвавшего команду –
(senderrank) показывает имя пользователя, вызвавшего команду, с упоминанием его ранга –
(senderrankonly) показывает ранг пользователя, вызвавшего команду –
(status) показывает текущий заголовок стрима –
(subscribers) показывает количество подписчиков канала (при наличии «партнёрки» или «компаньонки») –
(team_member_followers ?) показывает количество фолловеров заданного стримера из заданной команды (team_member_followers имя_команды, имя_пользователя)
(team_member_game ?) показывает текущую игру заданного стримера из заданной команды (team_member_game имя_команды, имя_пользователя)
(team_member_url ?) показывает ссылку на канал заданного стримера из заданной команды (team_member_url имя_команды, имя_пользователя)
(team_members ?) показывает список членов заданной команды (team_members имя_команды)
(team_name ?) показывает имя заданной команды (team_name имя_команды)
(team_random_member ?) показывает случайного члена указанной команды (team_random_member имя_команды)
(team_url ?) показывает ссылку на канал указанной команды (team_url имя_команды)
(titleinfo) показывает текущий заголовок стрима и его продолжительность –
(touser) показывает имя пользователя, вызвавшего команду, либо другого пользователя, в случае если оно указано после команды –
(uptime) показывает текущую продолжительность стрима –
(useronly=?) разрешает вызывать команду только заданному в теге пользователю (useronly=имя_пользователя)
(viewers) показывает текущее количество зрителей –
(views) показывает количество просмотров канала –
(writefile ?) записывает текст, указанный после команды, в заданный в теге текстовый файл (true – запись на новую строку, false – замена первой строки файла) (writefile имя_файла.txt, true, (echo))
(#) генерирует случайное число от 1 до 100 –
(# ?) генерирует случайное число из заданного в теге диапазона (# 1,250)
(1), (2), (3), … (9) подставляет указанное после команды (при её вызове) значение (или до 9 значений) –
(count) показывает количество вызовов команды (счётчик) –
(customapi ?) вызывает данные по заданной в теге ссылке API (customapi http://not.real.com/joke.php?name=$1)
(customapijson ?) вызывает данные в формате JSON по заданной в теге ссылке API (customapijson http://api.apixu.com/v1/current.json?key=ваш_ключ&q=$1&lang=ru {Температура в г. } location.name {: } current.temp_c { °C})
(echo) возвращает указанное после команды сообщение от имени бота (осторожно: с тегом также могут использоваться такие команды, как /timeout и /ban) –
(encodeurl ?) переводит пробелы в запросе, указанном после команды, в URL-кодировку (может использоваться внутри такого тега, как (customapi), а также сам содержать в себе такие теги, как (echo) и др.) (encodeurl https://google.com/search?q=(echo))
(follows) показывает количество фолловеров канала –
(game) показывает название текущей игры стрима –
(lasttip) показывает информацию о последнем донате –
(random) показывает случайное имя пользователя из чата –
(readfile ?) показывает первую строку из заданного в теге текстового файла, лежащего в папке «addons» бота (readfile имя_файла.txt)
(readfilerand ?) показывает случайную строку из заданного в теге текстового файла, лежащего в папке «addons» бота (readfilerand имя_файла.txt)
(@sender) показывает имя пользователя, вызвавшего команду, с символом «@» перед именем –
(sender) показывает имя пользователя, вызвавшего команду –
(setrole ?) назначает роль (setrole lirik, Стример)
(status) показывает текущий заголовок стрима –
(touser) показывает имя пользователя, вызвавшего команду, либо другого пользователя, в случае если оно указано после команды –
(uptime) показывает текущую продолжительность стрима –
(viewers) показывает текущее количество зрителей –
(writefile ?) записывает текст, указанный после команды, в заданный в теге текстовый файл (true – запись на новую строку, false – замена первой строки файла) (writefile имя_файла.txt, true, (echo))
