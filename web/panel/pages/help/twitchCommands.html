Базовые команды Twitch

/ban бан указанного пользователя
/block личная блокировка указанного пользователя
/clear очистка чата от всех сообщений для каждого пользователя в чате
/color окрашивание собственного никнейма указанным цветом
/commercial запуск рекламы на 30 секунд (опционально: с указанной длительностью 30, 60, 90, 120, 150 или 180 секунд)
/disconnect временное отключение чата
/emoteonly включение режима чата только для сообщений, состоящих из смайлов Twitch
/emoteonlyoff отключение режима чата только для сообщений, состоящих из смайлов Twitch
/followers включение режима чата только для фолловеров (опционально: с указанным стажем от 1 минуты до 3 месяцев)
/followersoff отключение режима чата только для фолловеров
/help справка по базовым командам Twitch
/host включение ретрансляции указанного канала (опционально: с указанным заголовком)
/ignore добавление указанного пользователя в личный игнор
/marker установка метки на понравившемся моменте стрима (опционально: с указанным описанием)
/me цветное сообщение
/mod присвоение роли модератора указанному пользователю
/mods список всех модераторов канала
/raid рейд указанного канала
/r9kbeta включение фильтра повторяющихся сообщений
/r9kbetaoff отключение фильтра повторяющихся сообщений
/slow включение медленного режима чата (опционально: с частотой 1 сообщение в указанные от 1 до 86400 секунд)
/slowoff отключение медленного режима чата
/subscribers включение режима чата только для подписчиков
/subscribersoff отключение режима чата только для подписчиков
/timeout таймаут указанного пользователя на 600 секунд (опционально: на указанный срок от 1 до 1209600 секунд и по указанной причине)
/unban снятие бана или таймаута с указанного пользователя
/unblock снятие личной блокировки указанного пользователя
/unhost отмена ретрансляции другого канала
/unignore снятие личного игнора с указанного пользователя
/unmod снятие роли модератора с указанного пользователя
/unraid отмена рейда
/untimeout снятие таймаута с указанного пользователя
/unvip снятие статуса VIP с указанного пользователя
/vip присвоение статуса VIP указанному пользователю
/vips список всех VIP-пользователей канала
/w личное сообщение указанному пользователю
